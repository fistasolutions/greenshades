## Overview

## How to Use This Repository

### For Non-Technical Staff

1. **Start with Week 1:** Open `00_introduction_to_ai_and_agentic_ai/00_what_is_ai.md`
2. **Read sequentially:** Each lesson builds on the previous one
3. **Complete exercises:** Try the "Try It" sections and check your answers
4. **Take quizzes:** Use the 5-question quizzes to validate understanding
5. **Bookmark cheat sheets:** Save the one-page cheat sheets for quick reference

### For Technical Staff (Developers, QA, DevOps)

1. **Follow the 8-week plan** but accelerate through Weeks 1-2 if you have AI background
2. **Focus on Week 4** (`03_mcp_and_specdriven_development/`) for hands-on implementation
3. **Deep dive into Week 3** (`02_ai_in_greenshades_ecosystem/`) for domain-specific use cases
4. **Reference `09_quick_reference_guides/`** daily for commands and templates

### For Leadership

1. **Start with Week 1** to understand AI fundamentals
2. **Prioritize Week 8** (`08_ai_leadership_and_strategy/`) for strategic planning
3. **Review Week 5** (`04_ai_ethics_and_security_basics/`) for governance frameworks
4. **Use Week 6** (`05_productivity_10x_framework/`) for ROI and OKR planning

Each lesson includes:
- **Quick checks:** Embedded "Try It" exercises with solutions
- **5-question quiz:** Mix of MCQ, True/False, and short answer
- **Applied mini-labs:** Hands-on exercises using Greenshades context
