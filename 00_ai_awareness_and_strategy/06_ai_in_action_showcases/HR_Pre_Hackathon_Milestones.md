# HR Department: Pre-Hackathon Milestones

**Department:** HR  
**Hackathon:** AI HR Automation & Employee Engagement  
**Duration:** 8 Weeks  
**Target Audience:** HR Managers, HR Specialists, Recruiters

---

## Overview

This 8-week pre-hackathon program prepares HR teams for the AI Agentic AI Hackathon by building foundational knowledge in AI-powered HR automation, recruitment, and employee engagement.

---

## Milestone 4: Hackathon Readiness (Weeks 7-8)

### Learning Objectives
- Review hackathon problem statements
- Form teams and assign roles
- Prepare HR tools
- Final skill validation

### Required Activities

**Week 7: Problem Statement Review**
- [ ] Review all 3 HR hackathon problem statements
- [ ] Research each problem domain
- [ ] Identify preferred problem to solve
- [ ] Prepare initial HR strategy

**Week 7: Team Formation**
- [ ] Form teams (3-5 people)
- [ ] Assign roles (HR lead, automation specialist, analyst)
- [ ] Set team communication
- [ ] Schedule team meetings

**Week 8: Tool Preparation**
- [ ] Set up HR AI tools
- [ ] Prepare HR templates
- [ ] Gather HR data (anonymized)
- [ ] Test tool integrations

**Week 8: Final Validation**
- [ ] Complete mock hackathon exercise
- [ ] Automate HR process for sample scenario
- [ ] Present to team for feedback
- [ ] Refine approach

### Assessment Checkpoint
- [ ] Team formed and roles assigned
- [ ] Tools prepared
- [ ] Problem statement selected
- [ ] Strategy prepared
- [ ] Mock exercise completed

**Readiness Criteria:** ✅ All preparations complete, ready for hackathon
