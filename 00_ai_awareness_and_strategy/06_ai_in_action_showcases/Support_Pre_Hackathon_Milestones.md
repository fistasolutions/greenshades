# Support Department: Pre-Hackathon Milestones

**Department:** Support  
**Hackathon:** AI Chatbot Development & Ticket Routing  
**Duration:** 8 Weeks  
**Target Audience:** Support Managers, Support Representatives, Customer Success

---

## Overview

This 8-week pre-hackathon program prepares Support teams for the AI Agentic AI Hackathon by building foundational knowledge in AI-powered chatbots, ticket routing, and knowledge base automation.

---

## Milestone 1: AI Fundamentals (Weeks 1-2)

### Learning Objectives
- Understand AI and agentic AI concepts
- Learn AI applications in Support
- Understand chatbot development with AI
- Complete foundational training

### Required Curriculum Modules
- ✅ `00_introduction_to_ai_and_agentic_ai/` (all 6 lessons)
- ✅ `01_ai_vs_ai_assisted_vs_ai_driven/` (all 4 lessons)
- ✅ `05_productivity_10x_framework/02_role_based_productivity_examples.md` (Support section)

### Hands-On Exercises

**Exercise 1: Support AI Tool Setup**
- [ ] Explore Support AI tools (chatbots, ticket routing)
- [ ] Set up chatbot development environments
- [ ] Test basic chatbot features
- [ ] Review chatbot examples

**Exercise 2: First AI Chatbot**
- [ ] Create simple chatbot
- [ ] Configure chatbot responses
- [ ] Test chatbot interactions
- [ ] Compare AI vs manual support

**Exercise 3: Understanding Support AI**
- [ ] Review Support AI use cases
- [ ] Understand chatbot workflows
- [ ] Study ticket routing with AI
- [ ] Identify Support AI opportunities

### Assessment Checkpoint
- [ ] Complete all curriculum modules
- [ ] Successfully create chatbot with AI

- [ ] Can explain chatbot workflows

**Readiness Criteria:** ✅ All exercises completed, all assessments passed

## Milestone 4: Hackathon Readiness (Weeks 7-8)

### Learning Objectives
- Review hackathon problem statements
- Form teams and assign roles
- Prepare support tools
- Final skill validation

### Required Activities

**Week 7: Problem Statement Review**
- [ ] Review all 3 Support hackathon problem statements
- [ ] Research each problem domain
- [ ] Identify preferred problem to solve

**Week 7: Team Formation**
- [ ] Form teams (3-5 people)
- [ ] Assign roles (support lead, chatbot developer, analyst)
- [ ] Set team communication
- [ ] Schedule team meetings

**Week 8: Tool Preparation**
