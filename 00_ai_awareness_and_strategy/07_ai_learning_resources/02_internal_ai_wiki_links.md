**Task:** Navigate internal resources to find:
1. Use case examples
2. Code examples
3. Best practices
4. Security guidelines

```
Resource Navigation:

1. Use Case Examples:
   - Wiki → Use Cases → Payroll and Tax
   - Find: "Payroll Anomaly Detection" use case
   - Review: Implementation details, results, lessons learned

2. Code Examples:
   - Git Repository → AI Code Examples → Anomaly Detection
   - Find: Sample code for anomaly detection
   - Review: Implementation patterns, MCP integration

3. Best Practices:
   - Wiki → Best Practices → Anomaly Detection
   - Find: Tips and patterns
   - Review: Testing strategies, performance optimization

4. Security Guidelines:
   - Wiki → Security and Compliance → Data Handling
   - Find: Payroll data security requirements
   - Review: Encryption, access controls, audit trails
```
